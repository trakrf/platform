# Stack: TypeScript + React + Vite

> **Package Manager**: pnpm
> **Test Runner**: Vitest
> **Build Tool**: Vite

## Lint
```bash
pnpm run lint --fix
```

## Typecheck
```bash
pnpm run typecheck
```

## Test
```bash
pnpm test
```

## Build
```bash
pnpm run build
```

## E2E Tests (Optional)
```bash
# If playwright.config.ts exists
pnpm run test:e2e
```
