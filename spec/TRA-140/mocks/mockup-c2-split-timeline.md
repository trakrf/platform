## Animation Suggestions

- **Selection**: Subtle scale + shadow on selected list item
- **Timeline load**: Fade in entries with stagger (50ms delay each)
- **Duration bars**: Animate width from 0 to final (200ms ease-out)
- **Panel transition**: Slide or fade in (mobile full-screen)

---

## Pros & Cons

### Pros
- Best of both: familiar layout + engaging visualization
- Duration patterns visible at a glance
- Persistent list for quick comparison
- Keyboard navigable
- Mobile-friendly with full-screen timeline view

### Cons
- More complex implementation (split panel + timeline)
- Narrower timeline width than full-page options
- Two components to maintain (list + timeline)
- May need to simplify timeline for narrow panels

---

## Technical Notes

### Layout
- CSS Grid: `grid-template-columns: 1fr 1fr` (50/50)
- Or Flexbox: `flex: 1` on each panel
- Responsive: Stack at < 768px

### Timeline Component
- Custom vertical timeline with flex layout
- Duration bars use percentage of max duration
- Connector lines: CSS `::before` pseudo-elements
- Entry stagger with CSS animation-delay or framer-motion

### State Management
- URL: `?asset=CAM-001&range=7d`
- Selected asset in local state or URL
- History fetched on demand
