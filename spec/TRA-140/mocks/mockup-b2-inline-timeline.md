## Pros & Cons

### Pros
- Visually engaging - easy to spot patterns
- Duration bars give instant relative comparison
- Modern, polished UX
- Differentiates from standard data tables
- Good for demos and stakeholder presentations

### Cons
- More complex to implement than table
- Scaling challenges for very long/short durations
- More vertical space needed
- May need table fallback for accessibility/export

---

## Technical Notes

- Timeline component: custom or use `react-vertical-timeline-component`
- Duration bars: CSS flexbox with percentage widths
- Animation: CSS transitions for expand/collapse, optional for bars
- Responsive: Stack on mobile, side-by-side info on desktop
- Lazy load history on expand
- URL state: `?expanded=CAM-001&range=7d`
- Consider framer-motion for polish
