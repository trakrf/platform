TRA-140: Asset Location Reports

Instance 1 Mockups (mocky directory)

Mockup A: Tabbed Reports - Current Locations

Standard table layout showing all assets with their current locations. Tab switcher to navigate between Current Locations and Asset History.

  • • Search and filter by location
  • • Sortable columns
  • • Click row to view asset history
  • • Consistent with existing TrakRF patterns
Table

Mockup B: Tabbed Reports - Asset History

Visual timeline showing an asset's movement history. Duration bars make it easy to see where the asset spent the most time.

  • • Vertical timeline grouped by day
  • • Duration bars proportional to time spent
  • • Date range filter
  • • CSV export
Timeline

Mockup C: Split Panel (Master-Detail)

Side-by-side view with asset list on left and history detail on right. Click an asset to instantly see its history without navigation.

  • • No tab switching needed
  • • Hybrid: compact table + visual timeline
  • • Fast exploration across multiple assets
  • • Best for power users
Split Panel

Design Approach Summary

These mockups explore the tabbed reports approach with a focus on the visual timeline for asset history. Mockup C combines both table and timeline in a split-panel layout for users who need to quickly compare multiple assets.
